# 04-16 會議摘要：結論先行框架、降低認知負荷、準備與傾聽

### 結論先行框架
以「核心訊息—支持論點—行動呼籲」輸出，確保高密度信息被迅速吸收。
- 先講結論再給理由與請求，縮短理解路徑並穩定決策節奏。
- 將結論與行動綁成一句，在打斷或急促環境下保持話語主導。
### 降低認知負荷
以少且精準對抗資訊傾倒與「知識的詛咒」。
- 只傳遞受眾在乎的可感價值與明確結果，刪除思考過程。
- 假設對方看不見你的腦內全景，提升明確度與可執行性。
###準備與傾聽
用事前推演與情緒承接，打造可持續的臨場掌控。
- 準備邏輯模塊而非逐字稿，預判尖銳問題以強化臨場自信。
- 先接住情緒再談方案，讓理性重啟避免邏輯失效。
- 以直述、故事、數據、情感多載體重複同一核心，對抗遺忘曲線。
- 每周練習只問問題不給建議，強化傾聽肌肉並重塑互動品質。
